# 文档索引

`docs/` 用于承载不适合直接放进 `README.md`、但又需要长期保留的补充说明。

当前建议入口如下：

- `模块系统.md`：解释 `MODULE_*` 宏、`linker.ld` 和模块生命周期机制
- `windows/`：Windows 构建相关的补充材料与历史参考
- `开发记录/`：保留阶段性记录，不作为主产品入口文档

如果新增文档，请先判断它属于：

1. **主入口说明**：应优先更新 `README.md` / `BUILD.md` / `README-WINDOWS.md`
2. **架构/规范**：应更新 `specs/`
3. **补充解释或历史记录**：再放入 `docs/`
